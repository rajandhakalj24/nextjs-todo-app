# Next.js Todo App

## Introduction  
This is a simple Todo application built using Next.js. It allows users to add, edit, delete, and mark tasks as completed.

## Features  
- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Mark tasks as completed  

## Prerequisites  
- Node.js (v12 or later)  
- npm (Node Package Manager) or yarn  
- Git  

## Setup Instructions  
1. **Clone the repository**  
   ```bash  
   git clone https://github.com/rajandhakalj24/nextjs-todo-app.git
   cd nextjs-todo-app  
   ```  

2. **Install dependencies**  
   ```bash  
   npm install
   # or  
   yarn install
   ```  

3. **Run the development server**  
   ```bash  
   npm run dev
   # or  
   yarn dev
   ```  
   Navigate to `http://localhost:3000` in your web browser to see the app.

## Usage Instructions  
- To add a new task, type the task name in the input box and click on 'Add'.  
- To edit an existing task, click on the 'Edit' button next to the task.  
- To delete a task, click on the 'Delete' button.  
- To mark a task as completed, click the checkbox next to the task.
